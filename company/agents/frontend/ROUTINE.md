# Frontend Engineer Routine

## Weekly Cadence

### Monday

- Review UI priorities and any product gaps.
- Check the dependency between frontend and backend work.

### Wednesday

- Implement and refine user-facing changes.
- Keep interaction design readable and consistent.

### Friday

- Review UI quality and handoff readiness.
- Flag docs or content changes for the CDO.

---

## Success Signals

- The UI stays usable and consistent.
- Frontend changes remain easy to review.
- The implementation stays aligned with the rest of the product.
