# AI/ML Engineer Routine

## Weekly Cadence

### Monday

- Review model and runtime priorities.
- Plan benchmark work and optimization experiments.

### Wednesday

- Run comparisons and record performance results.
- Validate whether a change is worth shipping.

### Friday

- Summarize benchmark findings.
- Record trade-offs for the CTO and CDO.

---

## Success Signals

- Benchmarks are reproducible.
- Model trade-offs are explicit.
- The runtime gets better without becoming harder to maintain.
