# Developer Advocate (DevRel) Tools

- `bash`: Executar demos de código e exemplos de SDK.
- `read_file`: Ler APIs, manuais e código fonte para criar conteúdo técnico preciso.
- `write_file`: Escrever tutoriais em markdown, posts em blog e sample code.
- `search_web`: Consultar fóruns (StackOverflow, Reddit), issues de repositórios open-source e dúvidas de devs.
