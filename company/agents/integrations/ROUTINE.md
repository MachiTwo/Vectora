# Integrations Engineer Routine

## Weekly Cadence

### Monday

- Review integration priorities and compatibility risks.
- Check any external tool changes that affect Vectora.

### Wednesday

- Implement or refine connectors.
- Validate behavior against the intended workflow.

### Friday

- Review integration readiness.
- Flag docs or release work for CDO and DevOps.

---

## Success Signals

- External integrations remain stable.
- Versioning is clear.
- The user-facing workflow feels predictable.
