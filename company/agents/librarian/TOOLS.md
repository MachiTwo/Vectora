# Librarian (Docs Engineer) Tools

- `bash`: Executar gerador de site Hugo, linting de markdown e deploy local.
- `read_file`: Revisar código-fonte para documentar comportamentos e arquivos `.pt.md` / `.en.md`.
- `write_file`: Escrever guias, tutoriais, tutoriais Mermaid e referências de API.
- `grep_search`: Buscar termos desatualizados, jargões sem explicação na pasta docs/.
- `list_dir`: Navegar pelos capítulos de documentação em Hugo.
